平台/代理商订单结算规则

  • 　平台入账 = ( 订单总额 - 订单退款 ) × {{rate}} %
  • 代理商结算 = ( 订单总额 - 订单退款 ) × (1-{{rate}} %)= 订单总额 - 订单退款 - 平台入账

店铺订单结算规则

  • 订单净营收 = 订单总额 - 订单总退款 - 运费
  • 抽取建设金 = 订单净营收 * 建设金抽取比例
  • 抽取广告费 = 订单净营收 * 广告费抽取比例
  • 　店铺结算 = 订单净营收 * ( 1 - 建设金抽取比例 - 广告费抽取比例 ) + 运费
